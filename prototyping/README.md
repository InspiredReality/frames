# Prototyping

Conversational three-page React web app.

## Pages

- **Play** `/` — free exploration, ideas, experimentation
- **Win** `/win` — goal-setting and strategy
- **Enjoy** `/enjoy` — reflection and celebration

## Getting started

```bash
cd prototyping
npm install
npm start
```

Opens at `http://localhost:3000`.

## Stack

- React 18
- React Router v6
- Pure CSS (no UI library)
